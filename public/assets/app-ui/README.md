# UI do app Gamellito (cópia local)

Estes PNGs foram copiados de `gamellito-app/assets` (projeto Unity / backup) para alinhar botões e HUD do jogo leve na web com o app:

- `UIFridgeButtonActive/Inactive` — botão geladeira (alimentar)
- `Glicosimetro.png` — medir glicemia
- `Seringa.png` — corrigir / insulina
- `app-exercise.png` — passar tempo (antes `Exercício.png`)
- `play.png` — próximo dia
- `heart.png`, `app-necessidade-full/vazio` — painel de vida / energia
- `app-mostrador-glicemia.png` — mostrador (antes `mostrador de glicemia.png`)
- `GamellitoFaceIcon.png` — ícone de humor
- `progress_bar_tex.png` — textura das barras

Para atualizar: substitua os ficheiros mantendo os nomes, ou edite `src/components/game-light/appUiAssets.ts`.
