# Minijogo estático (geladeira)

Alinhado ao modo **light** do site (`useFridgeGame`): abrir/fechar geladeira, lanches, contador de estrelas.

- **Fundo:** `room-bg.png` — cena da sala. Substitua mantendo o nome se quiser outra arte.
- **Interior:** usa `/assets/app-ui/UIFridgePageImage.png` (fallback para `geladeira.png`).
- Com **embed** (`NEXT_PUBLIC_GAME_MODE=embed`), `/jogos/experimente` carrega este `index.html` no iframe.
